# Tooltip Component Extension

This plugin allows the use of the MutableComponent type from RedWirePlatinum's 'Mutable Components' plugin with new procedures that fit alongside NerdyPuzzle's 'Item Tooltips' plugin.

This plugin depends on those two plugins, specifically versions:

- [MutableComponents](https://mcreator.net/plugin/114450/mutable-components) version 5.0.0, by [RedWirePlatinum](https://mcreator.net/user/693459/redwireplatinum)
- [Item Tooltips](https://mcreator.net/plugin/92306/item-tooltips) version 2.1, by [NerdyPuzzle](https://mcreator.net/user/703632/nerdypuzzle)

Furthermore, this plugin currently only supports MCreator 2026.2 and the NeoForge 26.1.2 generator.\
I do not plan on supporting older versions of either MCreator or NeoForge, but may add in new generators for new MCreator versions as they develop :)

I plan on keeping this plugin relatively simple as its purpose is simply to provide procedures that bridge RedWirePlatinum's 'Mutable Components' plugin wit hNerdyPuzzle's 'Item Tooltips' plugin.
